### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/logicalc-pro.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your API key:
   ```env
   API_KEY=your_actual_api_key_here
   ```
4. Start the app:
   ```bash
   npm run dev
   ```
